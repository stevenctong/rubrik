# computer

A collection of apps and scripts for Rubrik.

| Tool | Description |
|------|-------------|
| [`HTML-Apps/`](HTML-Apps/) | Standalone browser-based tools — open directly in a browser, no server required. Includes `csv-report-viewer.html` for viewing and filtering report CSVs with column type detection and Rubrik branding. |
| [`Get-Manage-CDM-Reports/`](Get-Manage-CDM-Reports/) | PowerShell script for managing custom reports on a Rubrik CDM cluster via the REST API — list, view, create, delete, and export. Outputs CSVs that can be loaded directly into the CSV viewer. |
